# 黒ひげ危機一髪

電子回路設計の授業で作りました。

## 使用した部品

| 名称 | 備考 |
| ------------- | ------------- |
| 自己保持ソレノイド  | CD07300240  |
| タクトスイッチ  | 赤、緑、黄  |
| LED  | 赤、緑、黄  |
| 抵抗  | 240Ω  |
